# 競合プラグインまとめ

## 一覧

- **Copilot for Obsidian** (6.0k⭐) - 総合型、Agent Mode、マルチモーダル
- **Smart Connections** (4.5k⭐) - セマンティック検索特化 + Smart Chat
- **Text Generator** (1.8k⭐) - テンプレート生成特化
- **ChatGPT MD** (1.3k⭐) - Privacy-First Tool Calling
- **Note Companion** (787⭐) - マルチモーダル整理
- **Infio Copilot** (626⭐) - インライン提案特化

---

## Copilot for Obsidian

**位置づけ**: 総合型AIアシスタント（最多機能）

**強み**
- Agent Modeの自律的ツール選択
- マルチモーダル対応（画像/PDF/YouTube）
- 大規模コミュニティ、商用モデル確立

**弱点**
- Plus版必須機能が多い
- ローカルファースト設計ではない

**学べること**
- Agent Modeの設計パターン
- Project Modeによる文脈管理

---

## Smart Connections / Smart Chat

**位置づけ**: セマンティック検索特化 + チャット分離

**強み**
- プライバシーファースト（ローカル埋め込み標準）
- Smart Environmentで基盤統合
- 大規模コミュニティ

**弱点**
- 生成・編集機能が弱い
- Smart Chatの認知度が低い

**学べること**
- セマンティック検索の実装
- プラグイン分離戦略

---

## Text Generator

**位置づけ**: テンプレート生成特化

**強み**
- 強力なテンプレートエンジン
- コミュニティテンプレート共有
- 多プロバイダー対応

**弱点**
- モーダル中心で即時性が弱い
- チャット機能がない

**学べること**
- Frontmatterによる設定設計
- テンプレート共有の仕組み

---

## ChatGPT MD

**位置づけ**: ノート内チャット特化（プライバシー重視）

**強み**
- 3段階承認のPrivacy-First Tool Calling
- マルチプロバイダー対応
- Per-Note設定

**弱点**
- インライン提案未成熟
- エディタ連動が弱い

**学べること**
- 承認フロー設計
- Frontmatterでの柔軟設定

---

## Note Companion

**位置づけ**: マルチモーダル整理特化

**強み**
- YouTube/音声/OCRの自動整理
- Inboxワークフロー

**弱点**
- 有料（月額）
- プライバシー懸念（クラウド寄り）

**学べること**
- 自動整理ワークフロー設計

---

## Infio Copilot

**位置づけ**: インライン提案特化

**強み**
- Cursor風ゴーストテキスト
- ローカル埋め込み標準搭載

**弱点**
- 認知度が低い
- ドキュメント不足

**学べること**
- CodeMirror統合のインラインUX
